# Sample Outputs

These files are checked-in examples of QuantSpec output artifacts generated from synthetic fixture demos. They are included for reviewer inspection only.

- `HYP-001-intraday-fail-demo/` shows a hypothesis closed by quality gates.
- `HYP-002-intraday-pass-demo/` shows a hypothesis marked as a candidate for human review.

The samples demonstrate the project flow:

```text
brief.yaml -> spec.md -> results.json -> gates.json -> report.md -> decision.md
```

They do not represent live trading performance or market edge.
